#### [20. 有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
```

**示例 4：**

```
输入：s = "([)]"
输出：false
```

**示例 5：**

```
输入：s = "{[]}"
输出：true
```

**提示：**

- 1 <= s.length <= 10^4
- s 仅由括号 '()[]{}' 组成

## 解题思路



```
class Solution:
    def isValid(self, s: str) -> bool:
        length = len(s)
        if length % 2 == 1:
            return False

        mapping = {
            ")": "(",
            "]": "[",
            "}": "{"
        }
        stack = []
        for i in s:
            if stack and i in mapping and mapping[i] == stack[-1]:
                stack.pop()
                continue
            stack.append(i)

        if stack:
            return False
        return True
```
